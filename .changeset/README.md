# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool
that works with multi-package repos, or single-package repos to help you version and publish your
code.

## Quick Start

**Create a changeset:**

```bash
pnpm changeset:add
```

**Create an empty changeset (for non-release changes):**

```bash
pnpm changeset:empty
```

**Check status:**

```bash
pnpm changeset:status
```

## Configuration Highlights

This project uses:

- **Linked packages** - All packages version together
- **GitHub changelog** - Automatic links to commits and PRs
- **Snapshot releases** - For testing pre-release versions
- **Prerelease support** - For alpha/beta/next releases
- **CI enforcement** - PRs require changesets (or empty changesets)
